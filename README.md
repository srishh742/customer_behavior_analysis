# customer_behavior_analysis
data analysis project showcasing customer behavior analysis using python ,power bi, sql
This project demonstrates an end-to-end data analytics workflow using the Customer Shopping Behavior dataset. It covers data loading, cleaning, exploratory data analysis (EDA), SQL-based business analysis, dashboard creation in Power BI, and reporting. The objective is to derive actionable business insights that help understand customer purchasing patterns and improve decision-making.

Dataset
Dataset: Customer Shopping Behavior
Records: 3,900 customers
Format: CSV

Age
Gender

Purchase Amount (USD)

Promo Code Used

Review Rating
Season
Frequency of Purchases
Location
Color
Size

Python – Data loading, cleaning, and EDA

NumPy – Numerical operations

PostgreSQL / MySQL / SQL Server – SQL analysis
Power BI – Interactive dashboard
Gamma – Project presentation (PPT)

Project Workflow
1. Data Loading
Imported the dataset into Python.
Verified data types and dataset structure.
Checked for missing values and duplicates.
2. Data Cleaning
Removed duplicate records.
Handled missing values where required.
Standardized column names and data formats.
Verified data quality before analysis.
3. Exploratory Data Analysis (EDA)
Analyzed customer demographics.
Studied purchase trends across categories.
Explored subscription behavior.
Compared revenue by age, gender, and category.
Visualized customer spending patterns.
4. SQL Analysis

Business questions answered using SQL:

Revenue generated by male vs. female customers.
Customers who used discounts but spent above average.
Top products by average review rating.
Standard vs. Express shipping purchase comparison.
Subscriber vs. non-subscriber spending analysis.
Highest revenue-generating categories.
Revenue by age group.
Purchase trends by season.
Customer segmentation.
Additional business insights.
5. Power BI Dashboard

Created an interactive dashboard including:

Total Customers
Average Purchase Amount
Revenue by Category
Subscription Status Distribution
Sales by Age Group
Purchase Trends
Interactive filters and slicers
6. Report & Presentation
Prepared a project report summarizing findings.
Created a presentation using Gamma to communicate insights effectively.
